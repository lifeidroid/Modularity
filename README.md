# **Modularity**
本开发框架模板本着：功能模块化、集成插件化、开发实用化、编码清晰化的原则。  
以MVP开发模式为基础，封装了实用的功能，如：TCP Server/Client的创建、经典蓝牙/低功耗蓝牙通信、Fragment间的通信以及各种自定义控件等。随着开发经验的增加也在不断更新完善中..
